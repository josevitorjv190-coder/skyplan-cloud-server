# SKYPLAN Cloud Server

Servidor WebSocket para multiplayer do SKYPLAN EFB.

## Deploy Railway
1. Suba esta pasta no GitHub
2. Conecte no Railway
3. Railway detecta automaticamente (Node.js, `npm start`)
4. Pegue o domínio gerado: `wss://SEU-PROJETO.up.railway.app`

## Health check
`GET /health` → `{"ok":true}`
